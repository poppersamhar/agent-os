页面功能总览

AgentOS 前端共 8 个一级页面，按 Sidebar 导航组织。

H
首页
Dashboard
数据看板 Member: Token消耗 / 活跃项目 / 任务成功率 · Admin: 全组织Token / 活跃成员 / 活跃项目
项目 Token 消耗概览 各项目 Token 用量排行与进度条
近7日消耗走势 柱状图展示周级 Token 趋势
项目上下文记忆 项目 context.md 列表，支持 @ 引用到 OrgAgent
OrgAgent 对话面板 企业效能管理智能体对话区，快捷操作入口
成员概览（Admin 专属） 成员列表、负责项目、本月 Token 消耗
全组织费用趋势（Admin 专属） 组织级周费用柱状图
S
Skills
技能资产
Skill 卡片网格 预设 Skill + 自建 Skill，按分类（知识工程/数据治理/分析/工具）展示
Skill Creator 新建 Skill 引导，配置名称/描述/图标/分类
Skill 编辑 / 删除 三点菜单：编辑 Skill 信息、删除自建 Skill
Skill 详情 查看 Skill 完整配置与使用说明
C
连接器
Connectors
连接器列表 MCP / API / 数字员工 / 其他 四类筛选展示
连接器卡片 名称、描述、分类、参数定义、状态
OrgAgent 管理面板 连接器统一管理与调度入口
K
知识库
Knowledge
全局知识图谱 可视化全组织知识资产关联关系（节点+连线）
知识源管理 全部 / 内部 / 第三方 / MCP / API 五类筛选
知识源卡片 名称、类型、描述、状态、条目数
OrgAgent 知识管理面板 知识库问答与知识源配置入口
P
项目
Project
任务列表 项目内所有任务卡片，支持置顶/重命名/归档
WorkAgent 对话面板 管理智能体对话区，发起任务、查看项目状态
文件 Tab 项目产物卡片置顶展示，上下文文件与产物分类管理
项目状态概览 当前任务执行状态、待审批提醒
T
任务执行
TaskSpace
聊天式执行流 8 种消息类型实时呈现（member/workagent/question/plan/subagent/completion/system/permission）
附件上传 + 号上传文件到任务上下文
Token 用量指示器 实时显示当前任务 Token 消耗
右侧面板 — 任务树 子任务拆分、执行进度与阻塞状态
右侧面板 — 项目树 项目全景、快速切换关联任务、引用历史上下文
N
新建任务
NewTask
任务名称输入 快速创建独立任务或项目内任务
快捷创建引导 推荐任务模板与最近使用
A
组织管理
Admin Only
成员管理 组织成员列表、权限分配
团队管理 团队架构与分组管理
用量管理 Token 用量统计与配额管理
系统设置 全局配置、安全策略、集成设置

Sidebar 导航结构

左侧 240px Sidebar，支持折叠。全局导航 + 项目树 + 独立任务历史。

01
主导航区
顶部固定导航 · 滚轮切换
  • 1.1
    首页（Home）
    个人/组织数据看板 + OrgAgent 对话区
  • 1.2
    Skills（Wrench）
    Skill 列表与 Skill Creator
  • 1.3
    连接器（Plug）
    MCP/API/数字员工连接器管理
  • 1.4
    知识库（BookOpen）
    全局知识图谱 + 知识源管理
02
任务操作区
快速创建 + 任务历史
  • 2.1
    新建任务（+）
    快速创建独立任务，进入 TaskSpace
  • 2.2
    历史对话（可折叠）
    独立任务列表，支持置顶/重命名/归档
03
项目管理区
我的项目 + 任务树
  • 3.1
    我的项目列表
    项目卡片，支持置顶/重命名/归档/展开折叠
  • 3.2
    项目内任务树
    展开项目后显示任务列表，点击进入 TaskSpace
  • 3.3
    新建项目（+）
    ProjectWizard 弹窗，填写名称与描述
04
组织管理区
Admin 专属 · 外部跳转
  • 4.1
    成员管理（UserCog）
    跳转外部管理后台
  • 4.2
    团队管理（Users）
    跳转外部管理后台
  • 4.3
    用量管理（TrendingUp）
    跳转外部管理后台
  • 4.4
    系统设置（Settings）
    跳转外部管理后台
05
用户卡片
底部固定 · 角色切换
  • 5.1
    用户信息展示
    头像、用户名、角色（Member/Admin）
  • 5.2
    切换角色
    Member/Admin 身份快速切换
  • 5.3
    退出登录
    清除登录态，返回登录页
06
全局交互
鼠标滚轮 · 快捷操作
  • 6.1
    滚轮切换全局页
    在首页/Skills/连接器/知识库间通过鼠标滚轮循环切换
  • 6.2
    Sidebar 折叠
    点击箭头收起为 56px 图标栏，再次点击展开

技术路线架构

从前端交互到后端服务，再到模型层与数据连接层的完整技术栈。

交互层 用户界面
React 18 + TypeScript Vite 构建工具 Tailwind CSS v4 Lucide React 图标 WebSocket 实时通信
应用层 核心逻辑
路由与状态管理 ThemeContext 主题管理 WorkAgentClient WS 客户端 Mock Data 数据层
智能体层 Agent 引擎
WorkAgentEngine 编排引擎 SessionManager 会话管理 SkillRegistry 技能注册 ToolRegistry 连接器注册
服务层 后端服务
NestJS + Express REST API 路由层 WebSocket 实时消息 pnpm + TypeScript
连接层 外部系统
MCP 协议对接 ERP / CRM / SCADA / OA 第三方数字员工接入 API 连接器（REST/GraphQL）
